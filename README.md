# netconf_project
## Requirements

* Python 3.5+
* setuptools 0.6+
* lxml 3.3.0+
* libxml2
* libxslt
* Django 3+
* djangorestframework 3+
* ncclient 0.6.13
* paramiko 3.3.1
* python-dotenv 1
* xmltodict 0.13.0

## Installation
Please create an virtual env and activate before installing above packages. use pip for installing the dependent packages.
